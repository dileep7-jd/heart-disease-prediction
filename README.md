This project predicts the likelihood of heart disease using machine learning techniques based on patient medical attributes such as age, cholesterol level, blood pressure, maximum heart rate, and other clinical parameters.
🚀 Features
Data preprocessing and cleaning
Exploratory Data Analysis (EDA)
Feature scaling and selection
Model training and evaluation
Comparison of multiple ML algorithms
🧠 Algorithms Used
Logistic Regression
Decision Tree
Random Forest
K-Nearest Neighbors
🛠️ Technologies Used
Python
NumPy
Pandas
Matplotlib & Seaborn
Scikit-learn
📊 Model Evaluation
Models were evaluated using:
Accuracy Score
Confusion Matrix
Precision & Recall
F1-Score
The best-performing model was selected based on overall performance metrics.
